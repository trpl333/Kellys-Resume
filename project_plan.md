# Kelly Peterson Portfolio — kellypetersonsped.com

## 1. Project Description
Professional personal portfolio website for Kelly Peterson, an Education Specialist (Special Education) with 26+ years of experience. Target audience: principals, SPED directors, and district HR. Goal: instantly communicate high-impact credibility, leadership, and student outcomes. Tone: confident, warm, factual. Privacy-safe.

## 2. Page Structure
- `/` — Home (Hero, Signature Impact, At-A-Glance, Impact Stories Preview, Leadership Preview, Credentials Strip, CTA)
- `/resume` — Resume (Timeline + Education + Download)
- `/impact-stories` — Impact Stories (Two STAR cards, privacy-safe)
- `/leadership` — Leadership (Chapman spotlight + committee list) [Phase 2]
- `/testimonials` — Testimonials (Placeholder template) [Phase 2]
- `/contact` — Contact (Email, location, download) [Phase 2]
